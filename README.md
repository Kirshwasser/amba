# Sagana Ambassadors – Лендинг амбассадорской программы

Самостоятельный одностраничный лендинг для рекрутинга блогеров в амбассадорскую программу Sagana. Отдельный репозиторий, не зависит от основного сайта.

## Что внутри

- `index.html` – основной лендинг с вертикальной прокруткой
- `presentation.html` – версия в виде презентации с горизонтальным перелистыванием слайдов (11 слайдов, навигация стрелками/клавиатурой/свайпом)
- `logo-light.svg`, `logo-dark.svg` – логотип Sagana для светлой и тёмной темы хедера
- `footer-logo.svg` – логотип в футере
- `hero-mockup.png` – мокап дашборда амбассадора в первой секции
- `privacy.html` – политика конфиденциальности (открывается из формы заявки)
- `images/` – папка под картинки секции «Почему Sagana продаётся» (`why-1.png` … `why-4.png`)

## Управление в `presentation.html`

- **Стрелочки внизу** или клавиши **← →** – переключают слайды
- **Пробел** / **Shift+Пробел** – следующий / предыдущий
- **Page Down / Page Up** – следующий / предыдущий
- **Home / End** – на первый / последний слайд
- **Свайп** на мобилке – слайды листаются как карусель
- **Mouse wheel** – листает слайды, когда внутренний контент слайда уже долистан до конца. Если внутри слайда много контента (FAQ, форма) – wheel прокручивает его вертикально, и только потом перелистывает на следующий слайд

Вверху видна тонкая прогресс-полоска по слайдам, в правом нижнем углу – счётчик «1 / 11».

## Стек

Никаких сборщиков и зависимостей – статический HTML.

- **Tailwind CSS** через CDN (`cdn.tailwindcss.com`) – кастом-конфиг с фирменным `#FF6B35`
- **Шрифт Inter** через Google Fonts
- Базовый JS внутри `<script>` для калькулятора и сабмита формы

## Локальный запуск

Можно открыть `index.html` напрямую в браузере. Для удобства разработки – лёгкий локальный сервер:

```bash
# Python 3
python3 -m http.server 8080

# Или Node
npx serve
```

Дальше открыть `http://localhost:8080`.

## Что нужно подготовить перед запуском в прод

| Что | Где править | Сейчас стоит |
|---|---|---|
| Картинки why-1…why-4 | `images/` | плейсхолдер «картинка 4:3» |
| ТГ-канал амбассадоров | футер, `t.me/saganaAmbassadors` | плейсхолдер |
| Ссылка «Презентация» | футер | `#` |
| Ссылка «База знаний и материалы» | футер | `#` |
| Сабмит формы | `<script>` в конце `index.html` | mailto:ambassadors@sagana.ru (MVP) |

Для прода форму стоит подключить к Tally / Formspree / собственному эндпоинту – сейчас сабмит просто открывает почтовый клиент.

## Структура комиссии (для контекста)

- **15%** с прямых рекомендаций – со всех платежей твоих рефералов
- **+10%** со 2-го уровня – с покупок рефералов твоих рефералов
- **+5%** с 3-го уровня – с покупок рефералов тех, кого привели твои рефералы

Все три уровня – пожизненно. Минимальный вывод 5 000 ₽.

## Деплой

Ничего особенного – статика. Любой провайдер: GitHub Pages, Vercel, Netlify, Cloudflare Pages, обычный nginx.

Например, для GitHub Pages:
1. Залить репозиторий на GitHub.
2. Settings → Pages → Source: deploy from branch (main) → root.
3. Получить URL вида `https://username.github.io/repo-name/`.

Желательно настроить кастомный домен типа `ambassadors.sagana.ru`.
